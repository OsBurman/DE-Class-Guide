# Exercise 10: Arithmetic and Comparison Operators

## Learning Objectives
- Practice arithmetic operators: +, -, *, /, %
- Practice comparison operators: <, >, <=, >=, ==, !=
- Understand operator results (numeric vs boolean)

## Your Task
Create a program demonstrating arithmetic and comparison operations.

## Requirements
1. Perform all arithmetic operations on two numbers
2. Perform all comparison operations
3. Show results clearly
4. Explain what each operator does

## Reflection Questions
1. What's the difference between / and %?
2. Why do comparison operations return true/false?
3. When would you use each comparison operator?

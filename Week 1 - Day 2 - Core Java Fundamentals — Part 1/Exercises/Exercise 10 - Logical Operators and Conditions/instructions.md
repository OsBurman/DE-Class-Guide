# Exercise 11: Logical Operators and Conditions

## Learning Objectives
- Practice logical operators: &&, ||, !
- Understand truth tables
- Combine conditions in real-world scenarios

## Your Task
Create a program demonstrating logical operators.

## Requirements
1. Demonstrate AND (&&) with truth table
2. Demonstrate OR (||) with truth table
3. Demonstrate NOT (!) 
4. Create a real-world condition combining multiple operators

## Reflection Questions
1. When would you use AND vs OR?
2. What does NOT do?
3. How do you combine multiple conditions?

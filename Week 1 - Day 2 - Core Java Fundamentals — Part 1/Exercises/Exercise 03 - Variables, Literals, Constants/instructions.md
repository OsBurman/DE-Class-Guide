## Key Concepts

**Variables**: Named storage locations that hold values
- Declaration: `int age;`
- Initialization: `age = 25;` or combined: `int age = 25;`
- Can be changed: `age = 26;`

**Literals**: The actual hard-coded values in code
- Integer literal: `42`
- Float literal: `3.14f`
- String literal: `"Hello"`
- Boolean literal: `true`
- Char literal: `'A'`

**Constants**: Variables that cannot be changed (using `final`)
- Declaration: `final int MAX_USERS = 100;`
- Convention: ALL_CAPS_WITH_UNDERSCORES
- Value cannot be modified: attempting to reassign causes compilation error

# Exercise 09: StringBuilder vs StringBuffer

## Learning Objectives
- Understand performance implications of string concatenation
- Practice using StringBuilder for efficiency
- Compare mutable vs immutable string approaches

## Your Task
Build a program comparing string concatenation with StringBuilder.

## Requirements
1. Build a string using + in a loop
2. Build the same string using StringBuilder
3. Compare performance (optional)
4. Explain the difference

## Reflection Questions
1. Why is StringBuilder faster than concatenation?
2. When would you use each approach?
3. What's StringBuffer?

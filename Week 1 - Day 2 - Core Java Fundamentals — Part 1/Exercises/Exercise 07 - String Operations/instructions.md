# Exercise 08: String Operations

## Learning Objectives
- Practice string searching and extraction
- Work with substring operations
- Replace string content
- Split strings into arrays

## Your Task
Create a program demonstrating advanced string operations.

## Requirements
1. Use indexOf() to find substrings
2. Use substring() to extract portions
3. Use replace() to modify strings
4. Use split() to break strings into arrays

## Reflection Questions
1. What's the difference between indexOf() returning -1?
2. Why is substring(0, 4) not the same as substring(0, 5)?
3. What happens when you split a string?
